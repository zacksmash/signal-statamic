### About Signal



Signal Statamic is a Statamic starter theme. Signal includes a modern workflow, with the latest build processes and is integrated tightly with UIkit CSS framework.



### Getting Started



To start a new Signal Statamic project, use the Tower CLI.



`tower new project-name`



Then, select Statamic from the menu. This will setup a new Signal Statamic project locally for you and begin the build process.
